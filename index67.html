Funcionamento de um computador

Inicialização do Computador

Quando você liga o computador, o processo de inicialização, conhecido como boot, é iniciado. A BIOS/UEFI é executada e carrega o sistema operacional.

Carregamento do Sistema Operacional

O sistema operacional, como o Windows, Linux ou macOS, controla o hardware e fornece uma interface gráfica para o usuário.

Inicialização do Sistema Operacional

Interface Gráfica

A interface gráfica do sistema operacional, como o desktop, é apresentada ao usuário. É aqui que você interage com o computador.

Abrindo um Programa

Para abrir um programa, o usuário clica em um ícone na área de trabalho ou no menu iniciar. O sistema operacional carrega o programa na RAM.Um exemplo é abrir o Word.
